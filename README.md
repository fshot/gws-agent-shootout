# GWS Agent Shootout

Structured comparison of AI agent tools for programmatic Google Workspace access (Docs, Sheets, Drive, Gmail, Calendar). Testing Claude Code, Codex CLI, Gemini CLI, and Claude Desktop against GWS CLI, community MCP servers, and native extensions.

**Status: scaffolding. No test results yet.**

This is the v2 follow-up to a [sloppy first attempt](https://cruxcapacity.com/blog/2026-04-14-ai-google-workspace-v1/) where I let Claude drive the research with no protocol. Raw v1 artifacts: [fshot/google-doc-experiments](https://github.com/fshot/google-doc-experiments).

See [CLAUDE.md](CLAUDE.md) for the full test protocol.
